# Instalação do ZuBlock

Este guia instala a versão descompactada do ZuBlock no Brave ou Chrome.

## Pasta pronta

A build local pronta para carregar está em:

```text
D:\ZuBlockMV3
```

## Brave ou Chrome

1. Abra `brave://extensions` no Brave ou `chrome://extensions` no Chrome.
2. Ative o `Modo do desenvolvedor`.
3. Clique em `Carregar sem compactação`.
4. Selecione a pasta `D:\ZuBlockMV3`.
5. Fixe o ZuBlock na barra do navegador para acessar o menu.

## Atualizar depois de uma nova build

1. Recrie a build do ZuBlock.
2. Substitua a pasta `D:\ZuBlockMV3`.
3. Volte em `brave://extensions`.
4. Clique no botão de recarregar do ZuBlock.

## Funções principais

- `Apagar agora`: remove temporariamente um elemento da página.
- `Salvar edição`: permite esconder partes do site de forma permanente.
- `Revisar salvos`: mostra e remove edições salvas do site atual.
- `Traduzir`: traduz visualmente o texto da página para português ou inglês.
- `Salvar tradução`: guarda traduções repetidas em cache local para acelerar visitas futuras.
- `Escudo Twitch`: ativa a proteção experimental para anúncios de livestream da Twitch.
- `Brave Clean`: atalhos para configurações e políticas do Brave.

## Observações

- A tradução vem desligada por padrão.
- A tradução atual suporta português e inglês.
- Páginas internas do navegador, como nova guia e configurações, não podem ser editadas por extensões.
- No Brave atual, forks Manifest V2 independentes podem ser marcados como incompatíveis; o ZuBlock carregável usa Manifest V3.
